# ☕ Java Arrays Practice Programs

A collection of beginner-friendly Java programs designed to understand the fundamentals of **Arrays** and **Methods** in Java. This repository contains hands-on examples that demonstrate array creation, manipulation, parameter passing, method overloading, varargs, array copying, and user input handling.

---

## 📌 Overview

This repository was created as part of my Java learning journey to strengthen my understanding of Core Java concepts through practical coding exercises.

The programs cover different ways of working with arrays, passing arrays to methods, returning arrays, deep copy vs shallow copy, anonymous arrays, method overloading, enhanced for loop, and more.

---

## 🚀 Topics Covered

- ✅ Array Declaration & Initialization
- ✅ Different Ways to Create Arrays
- ✅ User Input using Scanner
- ✅ Passing Primitive Variables to Methods
- ✅ Passing Arrays to Methods
- ✅ Returning Arrays from Methods
- ✅ Anonymous Arrays
- ✅ Returning Anonymous Arrays
- ✅ Method Overloading
- ✅ Variable Arguments (Varargs)
- ✅ Enhanced For Loop (For-Each Loop)
- ✅ Deep Copy vs Shallow Copy
- ✅ Final Arrays
- ✅ Common Array Exceptions
